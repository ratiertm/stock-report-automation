특정 TICKER 정보를 수집하려면? 

● 3가지 방법이 있습니다:

  1. API로 자동 다운로드 + 파싱 (권장)

● 패키지는 Python 3.14 (AppData)에 있고, PowerShell에서 uvicorn은 Anaconda Python을 사용 중입니다. PowerShell에서 아래
  명령어로 실행하세요:                                                                                                     
                                                                                                                             python -m uvicorn app.main:app --port 8000
                                                                                                                           
  또는 Anaconda에도 설치하려면:  
  
  # 서버 실행
  uvicorn app.main:app --port 8000

  # PLTR의 CFRA 리포트 다운로드 → 자동 파싱 → DB 저장
  curl -X POST "http://localhost:8000/api/fetch/PLTR?source=CFRA"

  # Zacks도
  curl -X POST "http://localhost:8000/api/fetch/PLTR?source=ZACKS"

  2. 워치리스트에 등록 → 일괄 수집

  # 워치리스트에 추가 (CFRA+ZACKS 동시)
  curl -X POST http://localhost:8000/api/watchlist/add \
    -H "Content-Type: application/json" \
    -d '{"ticker": "NVDA", "sources": "CFRA,ZACKS"}'

  # 워치리스트 전체 일괄 다운로드 + 파싱
  curl -X POST http://localhost:8000/api/fetch-watchlist

  3. 이미 PDF가 있으면 로컬 파싱만

  # 프로젝트 루트에 PDF 파일이 있으면
  curl -X POST "http://localhost:8000/api/parse-local/NVDA?source=CFRA"

  # 또는 파일 업로드
  curl -X POST http://localhost:8000/api/parse \
    -F "file=@NVDA-CFRA.pdf" -F "source=CFRA"

  수집 후 조회

  # 종목 정보 + 최신 리포트
  curl http://localhost:8000/api/stocks/PLTR

  # CFRA vs Zacks 비교
  curl http://localhost:8000/api/stocks/PLTR/compare

  # 블로그용 콘텐츠 변수 (30+ 필드)
  curl http://localhost:8000/api/content/PLTR

  # 재무 데이터
  curl http://localhost:8000/api/stocks/PLTR/financials
